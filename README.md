## Initial setup

1. Create virtual environment

```sh
python -m venv venv
```

2. Activate virtual environment

```sh
# shell
. venv/bin/activate

# powershell/cmd
.\venv\Scripts\activate
```

3. Install dependencies

```sh
pip install -r requirements.txt
```

4. Run program

```sh
python main.py
```

> To rerun the program, do only steps 2 and 4
